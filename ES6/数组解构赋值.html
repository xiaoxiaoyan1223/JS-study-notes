数组的解构赋值 1.1同时赋值多个变量 let [a,b,c,d]=[1,2,3,4] 1.2结构嵌套数组 const const arr = [1, [2, 3, [4, 5, 6]]]; const [a, [b, c, [d, e, f]]] = arr; a //1 b //2 c //3 d //4 e //5 f //6 1.3相同模式下的不完全解构 let [a, b, c] = [1, 2, 3, 4]; // 1 2 3 let [a, b, c, d] = [1, 2, 3]; // 1 2 3 undefined let [a, [b, c, [d, e]]] = [1, [2, 3, [4, 5, 6]]]; // 1 2 3 4 5 1.4解构的默认值 ES6 内部使用严格相等运算符（===），判断一个位置是否有值。所以，只有当一个数组成员严格等于undefined，默认值才会生效。 如果一个数组成员是null，默认值就不会生效，因为null不严格等于undefined 如果默认值是一个表达式，那么这个表达式是惰性求值的，即只有在用到的时候，才会求值 function f() { console.log('aaa'); } let [x = f()] = [1]; 根本不会在后台输出aaa

对象的解构赋值 1.1根据属性解构对象 对象的解构与数组有一个重要的不同。数组的元素是按次序排列的，变量的取值由它的位置决定；而对象的属性没有次序，变量必须与属性同名，才能取到正确的值。 let { foo: baz } = { foo: 'aaa', bar: 'bbb' }; baz // "aaa" let obj = { first: 'hello', last: 'world' }; let { first: f, last: l } = obj; // 起了个别名 f // 'hello' l // 'world' 对象解构和数组解构一样，都支持嵌套解构 对象解构同样支持默认值

函数参数的解构赋值 //普通 function fetch(option){ var name = option.name; var age = option.age; var like = option.like; } // ES6 大大简化了代码量 function fetch({ name, age, like }) { // 参数经历了 let { name, age, like } = option }

常见的使用场景 1.1 交换变量的值 let x = 1; let y = 2; [x, y] = [y, x]; 1.2 从函数中返回多个值 //返回一个数组 function example() { return [1, 2, 3]; } let [a, b, c] = example(); //返回一个对象 function example() { return { foo: 1, bar: 2 }; } let { foo, bar } = example();
